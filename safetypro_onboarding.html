Company profile

This information identifies your organisation on the platform.

Create projects

Add each active project. You can add more later from Admin.

Middle management

Add Cluster Heads, Sector Heads, Department Heads, and VPs.

Access scope: Tier 2 users see aggregated KPIs across multiple projects. Cluster Heads assigned to Project A, B, C see all three combined on their dashboard.

Project teams

Assign Project Heads, Area Heads, and Activity Heads to each project.

Scoping: Each user is restricted to the project (and area, if specified) you assign them to. A Project Head on Project A cannot see Project B data.

Site team

Add Supervisors, Engineers, Foremen, and Crew Heads. These users primarily use the mobile field interface.

Access scope: Site team members see only their assigned area. A Crew Head sees only their own assigned actions.
